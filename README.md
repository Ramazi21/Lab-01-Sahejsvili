## Исходный код
- Program.cs — основной файл вызова модулей. Выполняет основную навигационную работу в приложении. Является центром спроектированный архитектуры этого программного обеспечения
- Modules/InputModule.cs — модуль ввода данных. Обеспечивает наполнение данными для программы, которые позже будут обрабатываться.
- Modules/ValidationModule.cs — модуль проверки данных. Проверяет полученные данные из прошлого модуля на наличие ошибок и если есть уведомляет об этом пользователя.
- Modules/ProcessingModule.cs — модуль обработки и вывода данных. Последний модуль из представленных. Выполняет основную работу программы и затем отображает в пользовательском интерфейсе результат выполненной работы.

## Вспомогательные файлы
- README.md — подробное описание проекта, рассказывающее обо всёх нюансах программы. Имеет в себе описание каждого модуля
- .gitignore — исключения Git.

## Диаграммы
- UML-диаграмма модулей. Наглядное представление архитектуры приложения, с коректными связями между модулями. (используется из ЛР №2)
